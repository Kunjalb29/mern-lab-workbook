# 🚀 MERN Stack Lab Workbook

A complete collection of **MERN Stack Web Development Lab Experiments**, covering front-end, back-end, and database integration using **MongoDB, Express.js, React, and Node.js**.

This repository is structured experiment-wise and follows a **professional Git workflow with incremental commits**, making it ideal for academic submission and portfolio showcase.

---

## 📌 Repository Overview

This project contains step-by-step implementations of core MERN concepts:

* ⚛️ React Fundamentals (Components, Props, State)
* 🔀 Routing using React Router
* 🎨 UI Frameworks (Material UI, Bootstrap)
* 🌐 API Handling using Axios
* 🗄️ MongoDB & Database Integration
* 🔧 CRUD Operations
* 🚀 Backend Development with Node.js & Express
* 🔗 Full Stack Integration
