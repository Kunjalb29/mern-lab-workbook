# Experiment 2: Working with React Props and State

## Aim
To understand props and state in React.

## Description
Props are used to pass data between components.
State is used to manage dynamic data within a component.

## Pre-Lab Questions

1. What is state in React?
Answer: State is used to store and manage dynamic data.

2. Difference between state and props?
Answer: Props are read-only, state can be modified.

3. How to update state?
Answer: Using setState() or useState hook.

4. How to implement state?
Answer: Using useState() in functional components.

5. How to pass props?
Answer: By passing attributes to components.

Successfully implemented props and state in React.
