# Experiment 1: React Components

## Aim
To understand and implement functional and class components in a React application.

## Description
This experiment introduces the core building blocks of React applications, which are components. We will create custom reusable UI components.

## Pre-Lab Questions
1. **What is a React component?**
   **Ans:** A component is an independent, reusable piece of code that represents part of the user interface.

2. **What are the two main types of components in React?**
   **Ans:** Functional components and Class components.
