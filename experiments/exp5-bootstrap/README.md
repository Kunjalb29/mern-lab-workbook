# Experiment 5: Bootstrap

## Aim
To style a React application using Bootstrap.

## Description
This experiment involves integrating Bootstrap into a React project to leverage its grid system and pre-styled components.

## Pre-Lab Questions
1. **Why use Bootstrap with React?**
   **Ans:** Bootstrap provides a vast collection of responsive, mobile-first CSS components that speed up UI development.

2. **Can you use standard Bootstrap in React?**
   **Ans:** Yes, but react-bootstrap is often preferred as it replaces Bootstrap javascript with React components.
