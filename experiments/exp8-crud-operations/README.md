# Experiment 8: CRUD Operations

## Aim
To implement Create, Read, Update, and Delete operations using Node.js, Express, and MongoDB.

## Description
This experiment covers building RESTful APIs using Express to perform CRUD operations on a MongoDB database via Mongoose.

## Pre-Lab Questions
1. **What does CRUD stand for?**
   **Ans:** Create, Read, Update, Delete.

2. **Which HTTP methods correspond to CRUD?**
   **Ans:** POST (Create), GET (Read), PUT/PATCH (Update), DELETE (Delete).
