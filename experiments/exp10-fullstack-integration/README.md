# Experiment 10: Full-Stack Integration

## Aim
To integrate a React frontend, Node/Express backend, and MongoDB database into a complete MERN application.

## Description
The final experiment brings together all concepts to create a fully functional MERN stack application, featuring frontend and backend communication.

## Pre-Lab Questions
1. **What does MERN stand for?**
   **Ans:** MongoDB, Express.js, React, Node.js.

2. **How does the frontend communicate with the backend?**
   **Ans:** The frontend makes HTTP requests (e.g. using Fetch or Axios) to the backend APIs.
