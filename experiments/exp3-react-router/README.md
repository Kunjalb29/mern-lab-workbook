# Experiment 3: React Router

## Aim
To implement and understand routing in a React application using React Router.

## Description
This experiment demonstrates how to use react-router-dom to navigate between different pages in a single-page application without reloading the browser.

## Pre-Lab Questions
1. **What is React Router?**
   **Ans:** A standard library for routing in React. It enables the navigation among views of various components in a React Application, allows changing the browser URL, and keeps UI in sync with the URL.

2. **What is the difference between BrowserRouter and HashRouter?**
   **Ans:** BrowserRouter uses the HTML5 history API for clean URLs, while HashRouter uses the hash portion of the URL, which is useful for static file servers.
