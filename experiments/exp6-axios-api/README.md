# Experiment 6: Axios / API calls

## Aim
To fetch data from an external API using Axios in a React application.

## Description
This experiment demonstrates making asynchronous HTTP requests to a REST API using Axios and managing the fetched data with React state.

## Pre-Lab Questions
1. **What is Axios?**
   **Ans:** Axios is a promise-based HTTP client for the browser and Node.js.

2. **How does Axios differ from Fetch API?**
   **Ans:** Axios automatically transforms JSON data, has better error handling, and supports request interception and cancellation out of the box.
