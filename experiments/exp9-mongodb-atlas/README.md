# Experiment 9: MongoDB Atlas

## Aim
To connect a backend application to MongoDB Atlas.

## Description
This experiment introduces MongoDB Atlas, a fully-managed cloud database, and configures an Express app to connect to it.

## Pre-Lab Questions
1. **What is MongoDB Atlas?**
   **Ans:** MongoDB Atlas is a multi-cloud database service built by the same people that build MongoDB.

2. **Why use Atlas instead of local DB?**
   **Ans:** Atlas provides automated backups, scaling, and high availability without managing infrastructure.
