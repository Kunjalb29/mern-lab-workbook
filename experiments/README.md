# 🧪 Experiments – MERN Stack Lab

This directory contains all the **experiment-wise implementations** for the MERN Stack Web Development Lab. Each experiment is organized in a structured format to ensure clarity, reproducibility, and proper documentation.

---

## 📂 Structure of Each Experiment

Every experiment follows a consistent structure:

```bash
expX-topic-name/
├── README.md
├── code/
└── output/
```

### 📄 README.md

Contains:

* Aim
* Description
* Pre-Lab Questions & Answers
* Procedure
* Result

### 💻 code/

* Source code for the experiment
* React / Node / MongoDB implementation files

### 🖼️ output/

* Screenshots of execution
* UI outputs or terminal results

## 🔁 Development Workflow

Each experiment is developed using a **step-by-step Git workflow**:

1. Initialize experiment folder
2. Add documentation (README)
3. Implement code
4. Capture output
5. Commit each step individually

This ensures:

* ✔️ Proper version tracking
* ✔️ Clean commit history
* ✔️ Incremental learning
