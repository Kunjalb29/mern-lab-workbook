# MERN Lab Workbook

## Aim
To understand React components and how they are used.

## Description
React components are reusable building blocks of a user interface.

## Pre-Lab Questions

1. What are React components?
Answer: Components are reusable UI elements.

2. Functional vs Class components?
Answer: Functional are simple functions, class components use state and lifecycle.

3. How to create a functional component?
Answer: Using a JavaScript function returning JSX.

4. How to create a class component?
Answer: Using class extending React.Component.

5. Purpose of render method?
Answer: It returns JSX to display UI.
