# Experiment 7: MongoDB Basics

## Aim
To understand the basics of MongoDB and connect to it using Node.js.

## Description
In this experiment, we set up a connection to a local or remote MongoDB database using Mongoose in a Node.js environment.

## Pre-Lab Questions
1. **What is MongoDB?**
   **Ans:** MongoDB is a document-oriented NoSQL database program that uses JSON-like documents with optional schemas.

2. **What is Mongoose?**
   **Ans:** Mongoose is an Object Data Modeling (ODM) library for MongoDB and Node.js.
